<h1>Swagg Player</h1>
<h2>Cross browser javascript music player for the web! Works in IE 7,8,9, FF 3.5+, Chrome & Safari</h2>

<p>Note: The main JavaScript file - jquery-swagg-player.js passes JavaScript Lint (http://www.JavaScriptLint.com)</p>

<h2>Demos</h2>
<p>Examples are included in index.html. Production examples can be found at <a href="http://johnnyray.me">here</a> and <a href="http://swaggplayer.no.de/demos">here</a>.</p>

<p>API features will probably change until I reach version 1.0 so, when updating please read the change log and review the documentation to check to see if your code will be compatible with the new version.</p>
